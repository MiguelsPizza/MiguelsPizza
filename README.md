# Hey, I'm Alex

Founder of [**MCP-B**](https://github.com/WebMCP-org) (formerly WebMCP) — bringing the Model Context Protocol to the browser. I build the infrastructure that lets AI agents actually interact with web pages.

W3C [WebExtensions Community Group](https://github.com/w3c/webextensions) contributor. Based in Cincinnati.

## 2025–2026 has been a ride

Since January 2025 I've shipped **2,400+ contributions** across public and private repos — 924 commits, 281 pull requests, 18 new repositories. And that's just what GitHub counts. The private contribution graph tells an even bigger story (1,100+ restricted contributions on top of that).

I went from an idea to an open-source org with 20+ repositories, a Chrome extension, an npm package ecosystem, and a community of contributors. [WebMCP](https://github.com/MiguelsPizza/WebMCP) crossed **1,000 stars** along the way.

## What I've been building

### MCP-B / WebMCP Ecosystem
The core platform — a web standard for letting AI agents use tools directly in the browser. Think MCP, but native to the web.

| Project | What it does |
|---------|-------------|
| [**WebMCP**](https://github.com/MiguelsPizza/WebMCP) | The original repo that started it all. 1,024 stars, 72 forks. |
| [**npm-packages**](https://github.com/WebMCP-org/npm-packages) | Monorepo of `@mcp-b/*` packages — transports, React hooks, polyfills, type-safe SDK |
| [**chrome-devtools-quickstart**](https://github.com/WebMCP-org/chrome-devtools-quickstart) | Get AI agents controlling Chrome in minutes |
| [**webmcp-sh**](https://github.com/WebMCP-org/webmcp-sh) | Web-based MCP playground for testing servers and clients |
| [**mcp-ui-webmcp**](https://github.com/WebMCP-org/mcp-ui-webmcp) | Chat UI layer on top of MCP servers |
| [**examples**](https://github.com/WebMCP-org/examples) | Reference implementations and demos |
| [**Char**](https://github.com/WebMCP-org/char-ai-saas) | Embedded AI agent platform built on WebMCP |
