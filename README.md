## What I shipped

### MCP-B / WebMCP — browser-native Model Context Protocol

- [**WebMCP**](https://github.com/MiguelsPizza/WebMCP) — the original. 1,024 stars, 72 forks
- [**npm-packages**](https://github.com/WebMCP-org/npm-packages) — `@mcp-b/*` monorepo: transports, React hooks, polyfills, type-safe SDK
- [**chrome-devtools-quickstart**](https://github.com/WebMCP-org/chrome-devtools-quickstart) — AI agents controlling Chrome in minutes
- [**webmcp-sh**](https://github.com/WebMCP-org/webmcp-sh) — web-based MCP playground
- [**mcp-ui-webmcp**](https://github.com/WebMCP-org/mcp-ui-webmcp) — chat UI on top of MCP servers
- [**examples**](https://github.com/WebMCP-org/examples) — reference implementations and demos
- [**mcp-b**](https://github.com/WebMCP-org/mcp-b) — core browser-native MCP tools
- [**Char**](https://github.com/WebMCP-org/char-ai-saas) — embedded AI agent platform built on WebMCP
- [**char-plugin**](https://github.com/WebMCP-org/char-plugin) — Claude Code plugin for Char
- [**webmcp-userscripts**](https://github.com/WebMCP-org/webmcp-userscripts) — Tampermonkey scripts that give any site WebMCP tools
- [**char-scripts-template**](https://github.com/WebMCP-org/char-scripts-template) — starter repo for WebMCP user scripts
- [**WebMCP-declarative-example**](https://github.com/WebMCP-org/WebMCP-declarative-example) — declarative WebMCP exploration (Go)
- [**webmcp-agent**](https://github.com/WebMCP-org/webmcp-agent) — agent runtime for WebMCP
- [**big-calendar**](https://github.com/WebMCP-org/big-calendar) — React/Next.js calendar with WebMCP integration
- [**ai-tinkerers-webmcp-demo**](https://github.com/WebMCP-org/ai-tinkerers-webmcp-demo) — in-browser RAG demo (Transformers.js + Dexie)
- [**docs**](https://github.com/WebMCP-org/docs) — ecosystem documentation
- [**web**](https://github.com/WebMCP-org/web) — MCP-B demo site (full-stack, Postgres, real-time sync)
